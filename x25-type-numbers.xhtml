Note
ITU-T TSB defines the high order two bits of the first octet of call user
data as follows:

00 - Used for other ITU-T TSB recommendations (such as X.29)
01 - Reserved for use by "national" administrative
     authorities
10 - Reserved for use by international administrative authorities
11 - Reserved for arbitrary use between consenting DTEs
