Registration Procedure(s)
RFC required
Reference
[RFC1826][RFC2402][RFC2406][RFC4302][RFC4303]
Note
The Security Parameters Index is a 32-bit number used to identify the
security association of a datagram in certain security procedures
[RFC1826][RFC1827].  The first 256 SPI values are assigned by the IANA
and listed here.
