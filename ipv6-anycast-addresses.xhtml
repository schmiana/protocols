IPv6 Anycast Addresses

Registration Procedure(s)
IESG Approval
Reference
[RFC4291][RFC2526]
Note
IPv6 anycast addresses are defined in "IP Version 6 Addressing
Architecture" [RFC4291]. [RFC2526] further defines a set of
reserved subnet anycast addresses. These are the highest 128
interface identifier values, within each subnet.  The syntax
and rules for assigning new IPv6 reserved subnet anycast addresses
are defined in [RFC2526].  These addresses are assigned in descending
order.
