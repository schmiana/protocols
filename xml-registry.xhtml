IETF XML Registry

Last Updated
2015-03-02
Note
[RFC3688] defines an IANA maintained registry of XML documents used within IETF protocols.
These documents are divided into classes: XML Namespaces, Document Type Declarations, Schemas, and RDF Schemas.
Individual documents are then registered within each class. The class and document names are combined to form a name
which is then used within the 'urn:ietf:params:xml' portion of the IANA maintained URN space for protocol parameters.

This registry is intended for:

(1) identifiers whose use is described in an IETF specification, and

(2) identifiers that are allocated in IETF-controlled URI space, such as "urn:ietf:..." and/or "http://...ietf.org/..."

The registration template should cite the specification.

The list of classes is provided below.

Note
[RFC3688] defines an IANA maintained registry of XML documents used within IETF protocols.
These documents are divided into classes: XML Namespaces, Document Type Declarations, Schemas, and RDF Schemas.
The following are the entries for the 'rdfschema' class of documents.

This registry is intended for:

(1) identifiers whose use is described in an IETF specification, and

(2) identifiers that are allocated in IETF-controlled URI space, such as "urn:ietf:..." and/or "http://...ietf.org/..."

The registration template should cite the specification.
